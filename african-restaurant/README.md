# Safari Bites - African Restaurant Landing Page

A vibrant and welcoming landing page for Safari Bites, an African snack-style restaurant that celebrates authentic African cuisine and hospitality.

## 🎨 Design Features

### Color Palette
- **African Red** (#E74C3C) - Energy and warmth
- **African Yellow** (#F1C40F) - Sunshine and happiness  
- **African Green** (#27AE60) - Nature and growth
- Complemented by neutral tones for balance

### Typography
- **Primary**: Bebas Neue (bold, impactful headers)
- **Secondary**: Poppins (readable body text)

## 🔧 Customization Guide

### Changing Colors
Update the CSS variables in `style.css`:

```css
:root {
    --african-red: #E74C3C;
    --african-yellow: #F1C40F;
    --african-green: #27AE60;
    /* Add your custom colors here */
}
```

### Updating Menu Items
Edit the menu section in `index.html`:

```html
<div class="menu-item" data-category="snacks">
    <div class="menu-item-image">
        <img src="your-image.jpg" alt="Dish Name">
        <div class="menu-item-price">$9.99</div>
    </div>
    <div class="menu-item-content">
        <h3>Dish Name</h3>
        <p>Description of the dish</p>
    </div>
</div>
```

### Adding New Sections
1. Add HTML section in `index.html`
2. Add corresponding styles in `style.css`
3. Add JavaScript interactions if needed in `script.js`
